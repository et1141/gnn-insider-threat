# ADR Index

Architecture Decision Records capture technical decisions that affect behavior, reliability, or maintainability.

## Naming Convention

- File name: `NNNN-short-kebab-title.md`
- Example: `0001-training-stability-and-nan-mitigation.md`

## ADR Template

```md
# ADR NNNN: Title

## Status
Accepted | Proposed | Superseded

## Date
YYYY-MM-DD

## Context
What problem or constraint required a decision.

## Decision
What was decided.

## Consequences
- Positive impact
- Trade-offs
- Risks

## Validation
How we verified the decision.
```
